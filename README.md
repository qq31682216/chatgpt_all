# chatgpt_all
学习开源chatGPT类模型的指南，汇总各种训练数据获取、模型微调、模型服务的方法

# 说明
如果要搭建一个本地的chatGPT模型，需要借助开源的力量，大体上要解决两个主要问题：基础大模型用什么，怎么用指令微调(instruction)。开源项目众多，鱼龙混杂，这么一个索引式记录可以让大家少走弯路，欢迎贡献。
